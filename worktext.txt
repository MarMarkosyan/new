Порядок создания репозитория

1 создаём папку и заходим в ФАРе в папку
2 создаём репозиторий: git init
3 создаём папку на github.com с тем же именем
4 Создаем файлы в папке и добавляем эти файлы git add .
5 git commit -a -m "комментарий"
6 проверяем git status
7 связываем свой репозиторий с гитхаб:
git remote add origin https://github.com/L-graf/ПАПКА.git
8 командой git push --set-upstream origin master https://github.com/L-graf/ПАПКА.git создаём ветку Мастер, которая будет связана с нашей веткой Мастер фломастер



Порядок ежедневной работы
1. git pull проверка обновления
2. git status проверка состояния
3. git checkout master переход на ветку мастер
4. git checkout -b 'название' создание новой ветки
   git checkout <имя ветки> если нужная ветка уже существует, переход на нее
5. Работа в новой ветке
6. Сохранение работы
 git add . 
 git commit -a -m "комментарии"
7. Отправка изменений на гитхаб 
   git push --set-upstream origin <ИМЯ ВЕТКИ>
если в эту ветку уже пушили, команда
   git push
